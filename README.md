# DefineX üìñ
**DefineX** is a modern, responsive dictionary web application built with HTML, CSS, and Vanilla JavaScript. It leverages the Free Dictionary API to provide users with instant word definitions, phonetics, audio pronunciations, synonyms, and more.



## ‚ú® Features
* **Instant Search:** Get definitions, parts of speech, and examples for thousands of words.
* **Audio Pronunciation:** Listen to how words are pronounced using native audio files.
* **Voice Search:** Hands-free searching using the Web Speech API.
* **History & Bookmarks:** Keep track of your recent searches and save your favorite words for later.
* **Adaptive Theme:** Seamlessly switch between Light and Dark (Berry) modes.
* **Fully Responsive:** Optimized for all devices, featuring a desktop sidebar and a thumb-friendly mobile navigation bar.
